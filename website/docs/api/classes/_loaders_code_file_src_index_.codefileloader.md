This loader loads GraphQL documents and type definitions from code files
using `graphql-tag-pluck`.

```js
const documents = await loadDocuments('queries/*.js', {
  loaders: [
    new CodeFileLoader()
  ]
});
```

Supported extensions include: `.ts`, `.tsx`, `.js`, `.jsx`, `.vue`
